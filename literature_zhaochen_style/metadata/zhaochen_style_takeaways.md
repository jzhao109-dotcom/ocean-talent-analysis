# 赵晨论文风格与本研究借鉴要点

## 语言风格

赵晨等文章采用的是典型《中国软科学》管理学论文写法：开篇先嵌入国家战略语境，再指出现实治理问题，随后用已有研究不足引出本文贡献。语言重心不是炫示方法，而是围绕“何为人才、人才如何分化、人才何以成长”展开。

可借鉴到本研究的表达方式：

- 用“海洋强国建设”“战略科技力量”“关心海洋、认识海洋、经略海洋”等领域语境提出问题。
- 将研究对象表述为“海洋科技领军人才”或“中国海洋科技领域战略人才力量”，避免只说“文本样本”或“聚类对象”。
- 将聚类结果解释为“人才画像构型”“能力结构类型”“成才/贡献路径”，而不是单纯的方法输出。
- 结论写成“异质性与共性并存”：不同画像对应不同海洋赛道与能力组合，共性则落在海洋强国使命、自主创新、长期一线实践等要素。

## 整体逻辑

赵晨等文章的结构可概括为：

1. 国家战略需求提出问题：科技强国和企业创新主体需要科技领军人才。
2. 文献不足形成研究缺口：既有研究重单一画像、轻群体内部多元构型；重外显特征、轻内隐特质及成长机制。
3. 理论框架组织特征：以生涯资本理论统摄人力资本、社会资本、心理资本。
4. 数据识别人才画像：从公开生平文本中识别外显特征与内隐特质，再形成构型。
5. 案例解释成长路径：用典型人物材料解释“外部环境熏陶、内隐特质激活、外部能力涌现”的动态逻辑。
6. 贡献与启示回扣治理：理论上回应人才画像与成才机制，实践上服务分类培养和精准支持。

## 用到本项目的建议框架

本项目可对应改写为：

- 问题背景：海洋强国建设进入深海、极地、绿色低碳、智能装备和全球海洋治理并进的新阶段，亟需识别海洋科技领军人才的能力结构。
- 研究缺口：现有海洋人才研究更多关注培养政策、学科教育或单一能力要求，对海洋科技领军人才的多层能力结构、领域分化和精神驱动关注不足。
- 理论组织：以胜任力模型、人力资本资源理论、战略科技人才研究和海洋科学能力建设文献为基础，构建“蓝色胜任表征 - 协同创新支撑 - 深蓝精神内核”的三层框架。
- 结果表达：不要把结果写成算法标签，而要写成海洋事业场景中的人才构型，例如深海装备攻坚型、海洋观测预报型、蓝色生物资源转化型、近海治理与权益维护型等。
- 贡献表达：强调本文从公开人物文本中揭示了海洋科技领军人才“做什么、如何做成、为何坚持”的层级结构，补充了海洋人才研究中对微观能力构型的解释。

## 引文取向

主文献应少放方法文献，多放高质量领域文献：

- 战略科技人才、科技领军人才、战略科学家研究。
- 海洋强国、海洋科技创新、海洋科学能力建设。
- 海洋科研教育体系、国际大科学计划与海洋人才贡献。
- 胜任力、人力资本、生涯资本等少量基础理论。

方法文献只需在方法部分简要说明 BERTopic、UMAP、HDBSCAN 或 LLM 辅助编码的合理性，不宜成为全文引用主体。
