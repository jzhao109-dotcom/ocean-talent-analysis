# Zhao Chen Style Literature Pack

This folder collects shareable notes and reference metadata for adapting the argument style of Zhao Chen et al.'s paper on enterprise technology leading talents to the marine talent analysis project.

## Scope

- `metadata/zhaochen_style_takeaways.md`: writing style, argument logic, and transferable framing.
- `metadata/high_quality_domain_sources.csv`: high-quality domain and strategic talent references to prioritize.
- Local-only PDFs are intentionally ignored by Git because this repository is public and some source PDFs may be copyrighted or access-controlled.

## Working Principle

The target paper style is field-driven rather than method-driven. Methods should support the identification of talent profiles, while the main literature base should come from strategic talent, science and engineering talent development, marine science capacity, and ocean science systems.
