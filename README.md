# 海洋人才 · 多层能力聚类分析

基于 **209 位中国海洋科技领军人才** 的公开访谈与传记文本，对人才能力进行 BERTopic + UMAP + HDBSCAN 主题聚类，并用 K-Means 形成人才画像。本仓库同时保留 **两层** 和 **三层** 两套分析口径：

- **两层模型**（`valid_two_layer_archive/`）：外显行动 12 类 × 内隐驱动 7 类 → 5 类人才画像，结构精简，便于宏观对比。
- **三层模型**（`valid_three_layer_archive/`）：外层专业行动 14 类 × 中层方法支撑 6 类 × 内层使命价值 7 类 → 8 类人才画像，进一步显化"如何把事做成"的方法层。

## 三、关键发现

### 1. 内层使命是共同底色
**向海报国担当 / 海洋强国使命**和**自主创新 / 科技自立**几乎在所有画像簇中都有较高得分，是海洋科技领军人才共同的价值根基。两层模型中"向海报国担当"覆盖 359 人次为最高，三层模型中"海洋强国使命担当"覆盖 271 人次为最高。

### 2. 中层方法解释"差异化产出"
三层模型新增的**跨学科产学研转化能力（覆盖 460 人）**是覆盖率最高的中层能力，说明海洋科技领军人才普遍不局限于基础研究，而是强调从科学到产业的贯通。"如何把事做成"的方法层，是同样具备使命驱动的人才产出不同成果的关键变量。

### 3. 专业方向呈现海洋场景分化
无论两层（5 簇）还是三层（8 簇），人才画像都对应清晰的海洋场景：深海装备、近海治理、远航观测、生物资源、油气工程、医药保育、灾害预警……聚类对不同海洋赛道有较好的区分能力。

### 4. 小规模高战略价值类型
三层模型中"深水油气工程攻坚型"仅 9 人（4.3%），但集中承担能源安全和深水工程关键任务，体现"小规模、高专精、高战略性"的人才结构特点。

### 5. 双层 vs 三层选用建议
- 做宏观人才结构对比、传播展示 → 用**两层模型**（结构简洁，5 类画像便于讲述）
- 做精细能力诊断、人才培养路径设计 → 用**三层模型**（中层方法层显化，8 类画像更细粒度）

## 四、数据与方法

| 项目 | 说明 |
|---|---|
| 分析对象 | 209 位中国海洋科技领域领军人才 |
| 数据基础 | 公开访谈文本与传记材料 |
| 能力抽取 | DeepSeek API 按双层/三层结构抽取 signal_topics |
| 主题聚类 | BERTopic + UMAP + HDBSCAN，按层分别建模 |
| 人员聚类 | K-Means（轮廓系数辅助选 k） |
| 两层主题总数 | 12 + 7 = 19 |
| 三层主题总数 | 14 + 6 + 7 = 27 |
